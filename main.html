TCGA Toolbox

TCGA Toolbox provides a scripting environment to access, process, and analyze data from TCGA in the browser. It differs from other web apps in that the main interaction happens in the JavaScript Console. The base app can be extended by loading remote scripts.

This is a work in progress. The code is hosted on GitHub.

Tools

The tools are organized in the global TCGA namespace. The first argument of a callback function is reserved for an error object.

Load scripts

TCGA.loadScript(url, callback)

Get data

TCGA.get(url, callback)
TCGA.getRange(url, startByte, endByte, callback)
